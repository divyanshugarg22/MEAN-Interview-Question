## CSS Selectors
> element Selector = like <p>
> id Selector = write a hash (#) character
> class Selector = write a period (.) character
> Grouping Selectors
> group selectors, separate each selector with a comma.
    example 
    h1, h2, p {
    text-align: center;
    color: red;
    }
